
# Truffle Suite with Typescript

## Setup

### Install Truffle Suite

```bash
npm install -g truffle
```

### Create a new Project

```bash
truffle init
```
### Extension VSCode 

Install the extension with name:
```
Solidity by Juan Blanco
```

## Run

- Step 1
```bash
yarn migrate
```

- Step 2
```bash
yarn test
```
